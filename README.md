# 🌸 Iris Flower Classification using Machine Learning

## 📌 Overview
This project uses Machine Learning to classify Iris flowers into different species based on sepal and petal measurements.

The model predicts:
- Iris Setosa
- Iris Versicolor
- Iris Virginica
